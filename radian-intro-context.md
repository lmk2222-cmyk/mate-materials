# radian-intro.html — 작업 맥락 정리 (세션 이어가기용)

> 새 세션에서 이 파일을 이어 작업하기 위한 핸드오프 문서. 대상 파일: `radian-intro.html`

## 1. 사용자 / 작업 방식
- 한국 **고등학교 2학년 수학교사**. 학생용 **단일 HTML 교육자료**를 만든다(설치·인터넷 없이 더블클릭/링크로 열림).
- **작업 방식(중요)**: 바로 코딩하지 말고 **단계별로 함께 검토·확인 후** 진행. 모호하거나 결정이 필요한 부분은 임의로 정하지 말고 **선택지+추천안 제시 후 합의**. 응답은 한국어 존댓말.

## 2. 자료 목표
- 단원: 삼각함수 **호도법 도입**.
- 핵심 메시지: **단위원에서 "각이 자르는 호의 길이" = 각에 대응하는 하나의 실수.** → 호도법이 각을 실수에 대응시키는 도구임을 체득.
- **‘라디안’이라는 용어와 공식(ℓ=rθ, θ=ℓ/r)은 도입 단계(활동1)에서 일부러 미루고**, 탐구(동심원) 마지막에 **θ = ℓ/r** 로 정의·명명한다.

## 3. 파일 구조
- `radian-intro.html` 한 파일. `<script>` 안에 **독립된 IIFE 2개**:
  - **활동1** — canvas `#board`
  - **탐구(동심원)** — canvas `#board2`
- 순수 Canvas 2D + 포인터 이벤트. 외부 의존성 없음.

### 활동1 (단위원)
- 좌: 단위원 + 드래그 가능한 점 P / 우: **실수직선(정수 눈금)**.
- 점을 드래그하면 호의 길이가 실시간으로 우측 선분·숫자에 대응. **손을 떼는 순간**(또는 특수각 버튼/클릭) 호가 단위원에서 떨어져 **직선으로 굴러 펴지는 애니메이션**. (드래그 중엔 펴짐 없이 실시간 추적)
- 값 카드 2개: **각도(육십분법)**, **호의 길이 ℓ**. (라디안 카드·ℓ=rθ 공식은 제거)
- **초기 화면에서만** +x축에 주황 반지름 기준선 + "호의 길이 = 반지름의 길이" 라벨(합동 표시는 뺌). 조작하면 사라지고 ‘처음으로’로 복귀.
- 토글: **한 바퀴 넘기기**(최대 2바퀴=±4π), **음의 방향(시계)**. 기본 [0, 2π].
- 특수각 버튼(0~360°). 수업 포인트 안내는 **도형 상단**에 배치.

### 탐구(동심원)
- 동심원: 안쪽 r=1(**주황**, 고정 크기), 바깥 r(**청록**, 슬라이더 1.5~3 → 바깥 원만 커짐). 점 드래그로 공유 각.
- 우: 평행한 두 수직선에 두 호의 길이 비교(같은 각인데 바깥이 r배).
- **‘÷ 반지름 해보기’** 버튼 → 정규화 애니메이션: 바깥 막대가 줄어 두 값이 **θ로 일치**. 이때 바깥 막대·카드 라벨이 **"… ÷ 반지름"** 으로, 값은 ℓ/r 로 바뀜. **안쪽(첫) 막대는 나누지 않음**(÷1이라 의미 없음).
- 표는 **r, ℓ 두 열만**(ℓ÷r 열은 일부러 제거 → 학생이 ÷반지름으로 직접 발견).
- 정의 박스(÷반지름 누르면 등장): **θ = ℓ / r (호도법/라디안)**, + **(길이)÷(길이)라 단위가 약분되어 라디안은 단위를 안 씀** 설명 포함.

## 4. 색 / 표기 규칙 (확정)
- **양의 방향 = 주황(#f76707), 음의 방향 = 파랑(#1971c2)**, 원·반지름·중심각 = 회색.
- 우측 수직선은 **정수 눈금**(π 표기 안 함 — 도입 전이므로).
- "각도법" 용어 대신 **"육십분법"** 사용. 중심각 기호를 원 중심에 회색으로 표시.

## 5. 모바일 / 터치 최적화 (적용됨)
- 화면 폭 **< 640px**면 **세로 배치 자동 전환**(원 위 / 수직선 아래) — 활동1·탐구 둘 다. 캔버스 높이 자동 ↑.
- 좁은 화면 탭 영역 확대(미디어쿼리), `setPointerCapture`는 try/catch로 보호(임베드 환경 대비).
- 태블릿(≥640, iPad·안드로이드)은 기존 좌우 배치 유지.
